- **Python**: 3.10+

## System Requirements

### Minimum
- **RAM**: 8GB
- **CPU**: 4+ cores
- **Storage**: 10GB (for models)
- **Ollama**: Installed with at least one model

### Recommended
- **RAM**: 16GB
- **CPU**: 8+ cores
- **Storage**: 20GB
- **Models**: qwen2.5-coder:latest (4.7GB)

## Known Limitations

### Current Constraints
1. **Model Dependency**: Requires Ollama installation
2. **Hardware Requirements**: Significant RAM for larger models
3. **Internet**: Initial model download required
4. **Language Scope**: Limited to 6 programming languages

### Future Improvements (Post-v1.0.0)
1. **More Languages**: C++, Ruby, PHP support
2. **Enhanced UI**: VS Code extension
3. **Performance**: LRU cache eviction
4. **Deployment**: Docker containerization
5. **Internationalization**: English documentation
