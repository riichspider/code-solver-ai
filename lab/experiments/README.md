# Experiments

**Purpose**: Test new features, algorithms, and approaches in a controlled environment.

## Template for New Experiments

```markdown
# [Experiment Name]

**Date**: YYYY-MM-DD  
**Status**: [planning|running|completed|failed]

## Objective
Brief description of what you're testing.

## Hypothesis
What you expect to happen and why.

## Method
Steps to reproduce the experiment.

## Results
What actually happened.

## Conclusion
Was the hypothesis validated? What did you learn?

## Next Steps
Follow-up experiments or actions.
```

## Usage

1. Create a new folder for each experiment
2. Copy the template above
3. Document your findings objectively
4. Update status as you progress
